X-rays Classification


About the '/predict' Endpoint

At the '/predict' endpoint, you can use the API to classify X-rays of the chest. This feature allows you to analyze chest X-ray images for various purposes. Make sure to check the API documentation for more details on how to use this functionality.



About the '/apidocs' Endpoint

The '/apidocs' endpoint provides detailed information on how to use the API. Whether you are a developer or a user, the API documentation contains essential guidance on making requests, understanding responses, and utilizing the available features.
